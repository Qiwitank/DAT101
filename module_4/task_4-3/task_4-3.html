  1. [In this assignment, we use the DOM. Explain how this library can help us develop interactive web pages.]

    2. The DOM (Document Object Model) lets JavaScript interact with and modify the structure and content of web pages. It allows developers to dynamically change elements, respond to user input, and create interactive features without reloading the page.

  2. [What is an event? Simply explain what happens when we add an event listener to an object.]

    3. An event is an action or occurrence in the browser, like clicking a button or pressing a key. When we add an event listener to an object, we are telling the browser to run a specific function when that event happens on the object.

  3. [Sometimes we need to use an event object to get data from the event. Explain how this event object is structured, don't explain all functions or attributes.]

    4. The event object is automatically passed to event listener functions and contains information about what triggered the event. For example, in a keyboard event, event.key tells which key was pressed. In a mouse event, event.clientX and event.clientY give the position of the cursor. event.target refers to the element that caused the event.
